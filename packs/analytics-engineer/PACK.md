# Analytics Engineer

## Overview
Build the analytics infrastructure to understand exactly how users interact with your product — tracking events, analyzing funnels, measuring retention, and running experiments.

## What You Can Build
- Product analytics stack
- A/B testing framework
- Retention reporting system

## Workflow Breakdown
1. **Event Tracking** → Instrument user actions across surfaces
2. **Funnel Analysis** → Identify drop-off and conversion rates
3. **Retention Analysis** → Cohort-based engagement metrics
4. **A/B Testing** → Statistical experimentation framework
5. **Reporting** → Self-serve dashboards and automated alerts

## Outcome
A production analytics system with complete visibility into user behavior.
