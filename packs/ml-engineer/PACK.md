# ML Engineering Kit

## Overview
A complete ML engineering workflow: from data preparation and model training to evaluation, experiment tracking, and production serving.

## What You Can Build
- Classification/regression model
- Recommendation system
- ML-powered API endpoint

## Workflow Breakdown
1. **Data Prep** → Feature engineering, preprocessing, splitting
2. **Model Training** → Experiment tracking, hyperparameter tuning
3. **Evaluation** → Metrics, confusion matrix, bias audit
4. **MLOps** → Pipeline automation, model versioning
5. **Serving** → REST API, batch inference, monitoring

## Outcome
A production ML system with automated retraining and monitoring.
