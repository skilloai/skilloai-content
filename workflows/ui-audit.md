description: Evaluate frontend UI/UX for accessibility, responsiveness, and modern design.
---

1. Analyze the frontend component code (React, Vue, HTML/CSS).
2. Check for semantic HTML tags, ARIA attributes, and WCAG contrast compliance.
3. Apply responsive design principles (e.g., CSS Grid, Flexbox, media queries).
4. Suggest polished, modern styling touches (e.g., smooth transitions, glassmorphism).
